TeamCity plug-ins:
------------------

For plug-ins with pre-4.0 packaging:
- place the plug-in .jar file and any other dependent .jar libraries in:
    /usr/local/tomcat/webapps/teamcity/WEB-INF/lib

For newer plug-ins:
- place the plug-in .zip file in the /home/tomcat/.BuildServer/plugins directory

For further details see the notes at:
  http://confluence.jetbrains.net/display/TCD65/Installing+Additional+Plugins
