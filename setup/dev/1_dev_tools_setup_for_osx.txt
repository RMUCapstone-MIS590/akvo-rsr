Dev tools for Akvo development
------------------------------
1)  Install Xcode from the Mac App Store (so that we have gcc and other compilation tools available)

2)  Install MySQL 5.5 from: http://mysql.com/downloads/mysql
    (select the Mac OS X 10.6, 64-bit, DMG archive)

3)  Install Python 2.7 for Mac OS X: http://python.org/download/releases
    -> this will add /Library/Frameworks/Python.framework/Versions/2.7/bin or similar to your PATH environment variable

4)  Setup Git and Github access:
    a) Install Git command-line tools: http://git-scm.com/download
    b) Create a Github account for yourself at http://github.com
    c) Request Github access to the relevant Akvo repositories from one of the Akvo Dev team members
    d) Follow the setup notes at: http://help.github.com/mac-set-up-git
    e) Install the Github for Mac application: http://mac.github.com

5)  Install Homebrew: https://github.com/mxcl/homebrew/wiki/installation

6)  Install the following packages with Homebrew:
    a) brew install git-flow    # used as part of the Akvo dev process
    b) brew install imagemagick # used for building the Python Image Library (PIL)
